# FEAST ATCO Practice

Simulated aptitude test modules for Nav Canada FEAST Stage I & II air traffic controller selection.

> **Disclaimer:** For preparation use only. Not affiliated with Nav Canada.

## Tech Stack

- React 18 + TypeScript
- Vite
- React Router v7
- Tailwind CSS v4
- Framer Motion
- Zustand (with localStorage persistence)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```

The `vercel.json` rewrites all routes to `index.html` for SPA routing.
